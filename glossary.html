Glossary

  • Fundamental Type : A type that is recognized and/or handled specilly by the compiler or the language. Normally numbers , strings, booleans and arrays are fundamental types. The language literals noramlly only exist for fundamental types.

  • Primitive Type : A type that represents a value that is direclty handled in the machine stack and/or has native support in the machine. Normally numbers, booleans and arrays are primitive types. Primitive types are often also fundamental types and the compiler and/or the language handle them specilly.
